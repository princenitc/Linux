# Linux Commands

### Options
- -l -> shows the result in the listing format
- -r -> reverses the result
- -h -> show the result along with the size of the files  

All the options can be combined with any of the other irrespective of the order of the combination.


## Commands
- _history_ -> this will provide with the history of the desired commands.  If you find any command to be used from the hostory then you can call that by _!<command_number>_ this will run the particular command from the history table.
